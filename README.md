# MyForma - Workout Tracker (Server Deployment Guide)

Этот проект представляет собой Telegram Mini App для отслеживания тренировок с админ-панелью.

## Структура проекта
- `backend/`: Серверная часть (FastAPI)
- `frontend/`: Веб-интерфейс (HTML/JS/CSS)
- `forma.db`: База данных SQLite

## Установка на сервер

### 1. Подготовка окружения
Убедитесь, что установлен Python 3.9+.

```bash
# Клонируйте проект и перейдите в папку
cd myforma

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка переменных (.env)
Создайте файл `.env` в корне проекта (или отредактируйте существующий):
```env
TELEGRAM_BOT_TOKEN=ваш_токен
OPENAI_API_KEY=ваш_ключ
ADMIN_USERNAME=admin
ADMIN_PASSWORD=ваш_пароль_для_админки
SECRET_KEY=любая_секретная_строка_для_сессий
```

### 3. Запуск сервера
В этой версии запуск веб-сервера (FastAPI) и Telegram-бота (Aiogram) объединены. Вам больше не нужно запускать их отдельно.

**Команда для запуска всего приложения:**
```bash
python backend/main.py
```

Бот начнет слушать команды автоматически при старте сервера.

## Администрирование
Доступ к управлению базой данных осуществляется по адресу:
`http://ваш_ip_или_домен:8000/admin/`

Используйте логин и пароль, указанные в `.env`.
